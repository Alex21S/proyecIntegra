Reconocimiento de Texto para Formularios

Objetivo

Desarrollar una aplicación para reconocer cierta información en las imágenes. Mediante esta aplicación se pueden enviar fotografías a la base de datos o cargar alguna que ya exista en la base de datos independientemente de la extensión del archivo. El archivo de salida de la aplicación es en formato JSON con los campos que previamente se desea saber (ej. Nombre, domicilio, etc.)



Justificación

Tener una aplicación tecnológica e innovadora de manera gratuita que genere de forma eficaz y precisa la información filtrada para el llenado automático de formularios. Mediante esta aplicación se espera para facilitar la captura de la información para cualquier base de datos.



Descripción

El usuario escaneará u obtendrá una fotografía, posteriormente subirá la imagen mediante la aplicación. Cuando se encuentre la imagen en la base de datos, mediante el algoritmo de reconocimiento de texto “Tesseract” se procesará la imagen. En la siguiente etapa del proyecto, después de procesar la imagen deseada, se obtienen las coordenadas de la información que nos interesa saber para posteriormente genere un JSON con la información que el usuario dese saber.
